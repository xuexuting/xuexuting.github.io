客厅

客厅是家庭聚会、娱乐以及会客的重要空间，摆放在客厅的植物不仅要能够净化空气， 体态更要美观雅致，观赏起来才会感到心情愉悦。
客厅的角落中，适合摆放大型绿叶植物，比如幸福树、龟背竹、鹤望兰等，净化空气的同时增加了大气、爽朗的感觉。

幸福树

幸福树

幸福树花期长，花朵大，花香清淡。它的树形很美观，树姿优美，令人赏心悦目。

View

龟背竹

龟背竹

龟背竹在羽状的叶脉间,呈龟甲形散布许多长圆形的孔洞和深裂，茎有节似竹干，故名“龟背竹”。

View

鹤望兰

鹤望兰

鹤望兰叶子粗大而壮硕。花朵也非常美丽，整体呈现鸟头状，因此被称作天堂鸟。

View

散尾葵

散尾葵

散尾葵形态潇洒优美，枝条开张，枝叶细长而略下垂，株形婆娑优美，姿态潇洒自如。

View

卧室

卧室是睡觉休息的地方，追求宁静舒适的氛围，放在卧室的绿植最好能帮助睡眠。 气味也不能过于浓烈，熏得人头昏脑涨。
卧室的床头柜和窗台，适合摆放中小型绿植，比如吊兰、芦荟、绿萝等，保持空气湿度。

吊兰

吊兰

吊兰叶片细长柔软，抽生的小植株沿盆下垂，非常可爱。吊兰能吸收室内一定有害气体。

View

芦荟

芦荟

芦荟颜色翠绿，叶片层叠向上，易于栽种，是花叶兼备的多肉植物。部分品种的芦荟还可食用。

View

绿萝

绿萝

绿萝茎蔓细软、叶片娇秀，蔓茎常延伸可达20公尺以上。养殖难度低，也能吸收一定有害物质。

View

白鹤芋

白鹤芋

白鹤芋翠绿叶片，洁白佛焰苞，非常清新幽雅。对室内的氨气、丙酮、苯等有些许的清洁功效。

View

餐厨

餐厅的墙面上或者厨房的窗户上，养几盆颜色艳丽、体态美观的绿植，能使人心情愉快、也可增进食欲，烹饪时也更加享受。
一些中小型绿植，比如薄荷、空气凤梨、长寿花等，能很好的适应厨房空间。

冷水花

冷水花

冷水花适应性强，比较好养。株丛小巧素雅，叶色绿白分明，纹样美丽，可悬吊于窗前。

View

空气凤梨

空气凤梨

这是地球上唯一完全生于空气中的植物。它们品种繁多，形态各异，既能赏叶，又可观花。

View

长寿花

长寿花

长寿花植株小巧玲珑，株型紧凑，叶片翠绿，花朵密集。是非常好养的一种室内盆栽花卉。

View

薄荷

薄荷

薄荷全株青气芳香。养在厨房，可随之摘取叶子添加至于甜品饮料中，增加清凉的感觉。

View

卫生间

由于卫生间里异味、水汽较重以及不通风， 在马桶、洗手池旁可摆放一些体积小、耐阴、吸湿的植物。
虎皮兰、鸟巢蕨、孔雀竹芋等中小型品种是很好的选择，能让卫生间变得更健康，更美观。

虎皮兰

虎皮兰

虎尾兰叶片坚挺直立，叶面有虎尾状横带斑纹，姿态刚毅。虎皮兰耐阴喜湿，适应力很强。

View

鸟巢蕨

鸟巢蕨

鸟巢蕨叶子边缘稍微卷曲，葱绿而发亮，株形丰满，性喜高温湿润，不耐强光。

View

孔雀竹芋

孔雀竹芋

孔雀竹芋株型美观、叶面颜色五彩斑斓，具有较强的耐阴性，栽培管理较简单。

View

常春藤

常春藤

常春藤叶形美丽，四季常青，对环境适应力强，能一定程度上吸收空气异味。

View

阳台

阳台触承接阳光雨露，是一个房子里最适合摆放植物的地方。 适宜种植一些色彩明艳、喜日照的品种。
选择风信子、微型月季、蟹爪兰等植物，或悬挂、或靠近栏杆，能营造生机勃勃的氛围。

风信子

风信子

风信子植株低矮整齐，花序端庄，花色丰富，有怡人芳香。还可水培欣赏洁白舒展的根部。

View

微型月季

微型月季

微型月季是月季家族的新品种，植株矮小，花色奇异，花朵娟秀艳丽，花期全年均会开放.

View

蟹爪兰

蟹爪兰

蟹爪兰因节径连接形状如螃蟹的副爪，故得此名。花生于茎节顶端，红色或紫红色，极为美丽。

View

酢浆草

酢浆草

常春藤叶形美丽，四季常青，对环境适应力强，能一定程度上吸收空气异味。

View

办公场所

书房、办公室等办公场所应该是注意力需要高度集中的地方，摆放一些绿色植物， 不仅能改善环境，也能在工作劳累之时可以放松心情。
文竹、君子兰、铁线蕨等中小型植物，姿态舒展，叶色常青，是非常合适的选择。

文竹

文竹

文竹它的叶片轻柔纤细，密生如羽毛状，常年翠绿，枝干有节外形似竹，独具风韵。

View

君子兰

君子兰

君子兰叶片青脆挺拔，高雅端庄，是叶花果均美的名贵花卉，一直深受养花爱好者的喜爱。

View

铁线蕨

铁线蕨

铁线蕨茎叶形态优美，株型小巧，其淡绿色薄质叶片搭配着乌黑光亮的叶柄，格外优雅漂逸。

View

茉莉

茉莉

茉莉叶色翠绿，花色洁白，香气浓郁清雅，是最常见的芳香性盆栽花木。
