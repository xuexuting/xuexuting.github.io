推荐品种

这些植物养殖难度低，对光线的要求不高，是非常适合新手的植物。
不妨从它们入手，来打造一个生机勃勃的小花园。

吊兰

吊兰叶片细长柔软，从叶腋中抽生的甸甸茎长有小植株，由盆沿向下垂，舒展散垂；它既刚且柔，形似展翅跳跃的仙鹤，故古有“折鹤兰”之称。

View

芦荟

芦荟，原产于地中海、非洲，为独尾草科多年生草本植物。易于栽种，为花叶兼备的观赏植物。

View

薄荷

薄荷全株青气芳香。叶对生，花小淡紫色，唇形，花后结暗紫棕色的小粒果。多生于山野湿地河旁。

View

冷水花

冷水花适应性强，容易繁殖，比较好养，株丛小巧素雅，叶色绿白分明，纹样美丽，是相当时兴的小型观叶植物。

View

风信子

风信子植株低矮整齐，花序端庄，花色丰富，花姿美丽，是早春开花的著名球根花卉之一。

View

空气凤梨

这是地球上唯一完全生于空气中的植物，不用泥土即可生长茂盛，并能绽放出鲜艳的花朵。它们品种繁多，形态各异。

View

铁线蕨

铁线蕨茎叶形态优美，株型小巧，由于黑色的叶柄纤细而有光泽，好似少女柔软的头发，因此又被称为“少女的发丝”

View

文竹

文竹常年翠绿，枝干有节外形似竹，叶片纤细秀丽，密生如羽毛状，翠云层层，株形优雅，独具风韵，故又有云竹之称。

View

迷人的多肉

多肉植物颜色鲜艳、形态各异、娇小玲珑，总是会在第一眼就夺得人们的目光。
它们的名字也非常的富有诗意，静夜、花月夜、冰梅、广寒宫、晚霞……念来就像一首诗一般。
想发现更多多肉之美，不如去“特别专题”看看！

花月夜

叶子勺形，叶尖有红边，日照充足的情况下叶边会变红。 整株植物呈一朵莲花造型。花朵是铃铛形的，花色是黄色。花期春季。

View

乌木

叶呈莲座状紧密排列，叶面深绿如玉质感，叶缘及叶背上部在阳光充足的环境下展现尤为浓重的深棕色或黑棕色

View

墨西哥巨人

这种植物由于其特别的白色巨型叶子，而被人叫做墨西哥巨人，是一种大型多肉品种。厚叶子肉肉的，老叶子有时会偏粉。

View

苯巴蒂斯

莲座形态紧凑周正，叶片厚实，叶片底色为偏白的淡绿色，叶背具一条鲜明的红色短龙骨，与鲜明突出的红色叶尖相连，叶面分布着极细的斑纹。

View

冰莓

冰莓属于月影系多肉，在秋天的时候会变成桃红色，叶边有透明感，非常娇嫩可人，日晒下叶片会有果香味。

View

黛比

叶肉质，匙形，表面有薄粉，呈莲座状排列，一般呈淡紫色，在阳光充足且具一定温差的环境下叶片呈粉紫色。

View

紫罗兰女王

叶片呈莲座形密集排列，在强光下会变成梦幻的紫粉色，且叶缘有透明的冰边。

View

蓝石莲

莲座中小型，直径10-15厘米，灰绿色的叶片被有一层淡蓝色的霜粉，在阳光充足且具一定温差的环境下会变成粉红色。
